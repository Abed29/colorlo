# 🎨 Color Palette Generator

A modern, dynamic color palette generator built using **HTML, CSS, and JavaScript**.
Generate beautiful color combinations instantly and save your favorite palettes for later use.

---

## 🚀 Features

* 🎨 Generate random 5-color palettes
* 💾 Save palettes using browser local storage
* 📂 View saved palettes in a separate page
* ❌ Delete saved palettes anytime
* ⚡ Fast and fully dynamic (no backend required)
* 🎯 Clean and modern UI design

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **LocalStorage API**

---

## 📁 Project Structure

```
color-palette-app/
│
├── index.html      # Main palette generator
├── saved.html      # Saved palettes page
├── style.css       # Styling
├── script.js       # Generator logic
└── saved.js        # Saved palettes logic
```

---

## ⚙️ How It Works

1. Click **Generate** to create a new color palette
2. Click **Save** to store the palette
3. Open **Saved Palettes** to view stored palettes
4. Delete palettes when no longer needed

All data is stored locally in your browser using **localStorage**.
